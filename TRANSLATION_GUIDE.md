# Persian Translation Guide

This file documents the translation mapping from Arabic UI text to Persian.

## Common Translations

### Authentication
- تسجيل الدخول → ورود
- إنشاء حساب جديد → ایجاد حساب جدید
- البريد الإلكتروني → ایمیل
- كلمة المرور → رمز عبور
- تأكيد كلمة المرور → تأیید رمز عبور

### General
- عرض → مشاهده
- تعديل → ویرایش
- حذف → حذف
- إضافة → افزودن
- حفظ → ذخیره
- إرسال → ارسال
- تأكيد → تأیید
- إلغاء → لغو
- جاري → در حال
- حدث خطأ → خطا
- مطلوب → الزامی
- غير مصرح → غیرمجاز
- موجود → موجود
- نجاح → موفقیت

### Exam Related
- الامتحان → آزمون
- السؤال → سوال
- الأسئلة → سوالات
- الإجابة → پاسخ
- الإجابات → پاسخ‌ها
- صحيح → صحیح
- خطأ → غلط
- غير مجاب → بدون پاسخ
- الشرح → توضیح
- المدة → مدت زمان
- دقيقة → دقیقه

### Status
- نشط → فعال
- غير نشط → غیرفعال
- مكتمل → تکمیل شده
- قيد التنفيذ → در حال انجام
- منتهي → منقضی شده

### Admin
- إدارة → مدیریت
- المستخدمين → کاربران
- المحتوى → محتوا
- التذاكر → تیکت‌ها
- التقارير → گزارش‌ها

### User
- المحاولات → آزمون‌ها
- الأخيرة → اخیر
- النتيجة → نتیجه
- السابقة → قبلی
